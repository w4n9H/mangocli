About

A lightweight AI coding assistant running directly in your terminal. Inspired by Claude Code, built with zero external dependencies — just the Python standard library.

Mangopi CLI brings AI-powered coding workflows to your terminal with instant startup, autonomous file editing, shell execution, and context-aware conversation management. No Electron, no Docker, no heavy frameworks.

Features
Zero Dependency Python standard library only — no pip install hell
Instant Startup Launches in milliseconds, no heavy framework loading
Tool Calling Read, write, edit, search, grep, bash — fully autonomous
Context Management Auto-compacting, session persistence, memory injection
Terminal UX Claude Code–style interface with spinner, diff, token stats
Fully Hackable Single file, easy to read, modify, and extend
Installation

From PyPI:

$ pip install mangopi-cli
$ mangopi-cli

Or from source:

$ git clone git@github.com:w4n9H/mangopi-cli.git
$ cd mangopi-cli
$ python mangopi_cli.py
Configuration
# Set your API endpoint
$ export MANGO_KEY="your_api_key"
$ export MANGO_API_URL="https://api.deepseek.com/chat/completions"
$ export MANGO_MODEL="deepseek-v4-flash"

# Optional
$ export MANGO_MAX_CONTEXT=1000000
$ export MANGO_LANG=zh
