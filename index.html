// FEATURES // 特性

⚡

Zero Dependencies 零依赖

Pure Python stdlib — urllib, json, subprocess. No pip install required. Single .py file. 纯 Python 标准库 — urllib, json, subprocess。无需 pip 安装，单个 .py 文件。

🔄

Agentic Loop Agentic 循环

Autonomous tool-calling loop: read → think → edit → verify. Multi-step reasoning without manual intervention. 自主工具调用循环：读取→思考→编辑→验证。多步推理无需人工干预。

📦

Smart Context 智能上下文

Three-tier compact: micro (expire stale tool results), session-memory (retain recent), fallback (strip + trim). Auto triggers at 80%. 三级压缩：micro（过期工具结果）、session-memory（保留近期）、fallback（剥离+裁剪）。80% 阈值自动触发。

🛡️

Safety Checks 安全检查

Dangerous command detection (rm -rf, sudo, chmod 777, mkfs...). User confirmation before execution. Path sandboxing. 危险命令检测（rm -rf, sudo, chmod 777, mkfs…）。执行前需用户确认。路径沙盒。

💾

Session Persist 会话持久化

Sessions saved as JSON in .mangocli/session/. Corrupt files auto-backup. Resume where you left off. 会话以 JSON 保存在 .mangocli/session/。损坏文件自动备份。断点续聊。

🌐

i18n Ready 国际化

Built-in Chinese & English. Set MANGO_LANG=zh for Chinese UI output. System prompt adapts automatically. 内置中英文。设置 MANGO_LANG=zh 切换中文 UI 输出。System Prompt 自动适配。

// PROVIDERS // 模型接入

DeepSeek

Auto-detect deepseek models. Enables thinking/reasoning mode via extra_body. 自动检测 deepseek 模型，通过 extra_body 启用 thinking 推理模式。

DEFAULT 默认

OpenAI

Full OpenAI-compatible API. Supports any /v1/chat/completions endpoint. 完整 OpenAI 兼容 API。支持任何 /v1/chat/completions 端点。

Any Compat 任意兼容

Qwen, Moonshot, etc. — as long as it speaks OpenAI protocol. Qwen、Moonshot 等 — 只要支持 OpenAI 协议即可接入。

// CONFIG // 配置

  • MANGO_KEY API key (required) API 密钥（必填）
  • MANGO_API_URL Endpoint (default: deepseek) 端点（默认：deepseek）
  • MANGO_MODEL Model (default: deepseek-v4-flash) 模型（默认：deepseek-v4-flash）
  • MANGO_MAX_CONTEXT Max tokens (default: 1M) 最大 token 数（默认：1M）
  • MANGO_LANG Language: en / zh (default: en) 语言：en / zh（默认：en）
export MANGO_KEY="your-key"
export MANGO_API_URL="https://api.deepseek.com"
export MANGO_MODEL="deepseek-v4-flash"

# optional# 可选
export MANGO_MAX_CONTEXT="1000000"
export MANGO_LANG="zh"
